The Menu Component is a Navigation drawer that slides in from the side of the current view. By default it slides from left but that can also be over ridden.

Menu will be displayed differently based on mode. However display type caan be changed to any of the available menu types

The menu element should be a sibling to the app's content element

There can be any number of menus attached to the content. These can be controlled from the templates or programatically using the MenuController

Click on the menu icon on the lop left to see the sliding menu options

Click on the button below to see the same action as that of the menu icon button of top left region.

Click on the 'Close' item on top right to close the menu and go to the list of UIComponents
